Terminal Based

Experience SomaFM radio directly in your terminal with an intuitive text-based interface.

Stream Buffering

Built-in buffering ensures uninterrupted playback even with unstable connections.

Multiple Stations

Access all your favorite SomaFM stations from ambient to electronic genres.

Get Started

Clone the Repository

Get the source code from GitHub:

git clone https://github.com/zsh-ncursed/somafm_tui.git
cd somafm_tui

Install Dependencies

Make sure you have the required dependencies:

# Example dependencies (adjust based on actual requirements)
sudo apt install libmpv-dev libcurl4-openssl-dev

Build and Run

Compile and start enjoying SomaFM:

make
./somafm_tui

Experience SomaFM Differently

Enjoy your favorite internet radio stations without leaving the comfort of your terminal.
